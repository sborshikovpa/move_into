# 01. Основы Python

## Главное

Python выполняет выражения, хранит результаты в переменных, работает с разными типами данных и позволяет управлять ходом программы через условия, циклы и функции.

Минимальный цикл мышления в ноутбуке: написать маленький кусок кода, запустить, посмотреть результат, изменить, повторить.

## Вывод, ввод и комментарии

```python
print("Hello world")
print("Hello", "Grisha")

name = input()
```

- `print()` выводит значения.
- `input()` читает строку от пользователя.
- `#` делает однострочный комментарий.
- Тройные кавычки часто используют как многострочный текст, но для обычного комментирования кода лучше `#`.

## Операторы

### Арифметика

```python
2 + 2
2 * 3
4 / 2     # обычное деление, результат float
5 // 2    # целочисленное деление
5 % 2     # остаток от деления
2 ** 3    # степень
```

### Сравнения

Сравнения возвращают `True` или `False`.

```python
2 + 2 == 4
3 != 2
3 > 2
3 <= 3
```

### Логика

```python
True and False
True or False
not True
```

Порядок выполнения: сначала арифметика, затем `not`, потом `and`, потом `or`, затем сравнения и проверки вроде `in`. Скобки меняют порядок явно.

```python
(True or False) and not True
```

## Встроенные функции

```python
round(1.6)
sum((3, 2))
min([1, 2, 3, 5])
max((-100, -50, -3))
len((2, 3, 4, 5))
sorted([1, 5, 2, 4], reverse=True)
```

Функции вызываются как `name(arguments)`. Некоторые принимают дополнительные именованные аргументы, например `reverse=True`.

## Переменные

```python
a = 4
name = "Grisha"

b = 2
b += 1
b -= 3
```

Слева от `=` должно быть имя переменной, справа может быть выражение. Присваивание выполняется после вычисления правой части.

Полезный прием для обмена значениями:

```python
a = 10
b = 20
a, b = b, a
```

## Типы данных

Основные типы из урока:

- `str`: строка.
- `int`: целое число.
- `float`: дробное число.
- `bool`: `True` или `False`.
- `NoneType`: отсутствие значения, записывается как `None`.

```python
type("строка")
type(2342)
type(223.0)
type(True)
type(None)
```

Приведение типов:

```python
float(2342)
float("123.1")
int("123")
int(12.2)
str(5)
```

Не любую строку можно превратить в число: `int("hello")` упадет с ошибкой.

## Коллекции

### Список `list`

Список хранит упорядоченный набор элементов и создается через `[]`.

```python
items = [1, 2, 3, "text", None]
items.append("new")
last = items.pop()
```

Полезные операции:

- `append(x)`: добавить элемент в конец.
- `pop()`: удалить и вернуть последний элемент.
- `index(x)`: найти индекс элемента.
- `sort()`: отсортировать список на месте.

Сортировать вместе числа и строки нельзя, потому что Python не знает, как сравнивать разные типы.

### Индексация и срезы

```python
items = ["a", "b", "c", "d"]
items[0]      # "a"
items[-1]     # "d"
items[1:3]    # ["b", "c"]
items[:2]     # ["a", "b"]
items[2:]     # ["c", "d"]
```

Индекс начинается с `0`. В срезе правая граница не включается.

### Словарь `dict`

Словарь хранит пары `ключ: значение`.

```python
person = {
    "name": "Bob",
    "age": 30,
}

person["city"] = "Moscow"
person["name"]
```

Полезные методы:

- `keys()`: ключи.
- `values()`: значения.
- `items()`: пары ключ-значение.
- `get(key)`: безопасно получить значение, если ключа может не быть.

### Многоуровневые коллекции

Можно вкладывать списки и словари друг в друга.

```python
data = {
    "users": [
        {"name": "Ann"},
        {"name": "Bob"},
    ]
}

data["users"][1]["name"]  # "Bob"
```

## `in`

Оператор `in` проверяет наличие элемента.

```python
"a" in "cat"
2 in [1, 2, 3]
"name" in {"name": "Bob"}
```

Для словаря `in` проверяет ключи.

## Указатели и изменяемость

Переменная не "содержит объект внутри себя", а ссылается на объект. Из-за этого две переменные могут указывать на один и тот же список.

```python
a = [1, 2]
b = a
b.append(3)

a  # [1, 2, 3]
```

Чтобы получить отдельный список, нужна копия:

```python
b = a.copy()
```

## Условия `if`

```python
number = 10

if number % 2 == 0:
    print("четное")
else:
    print("нечетное")
```

Блок кода определяется отступом. Обычно используется 4 пробела.

## Циклы

### `for`

`for` перебирает элементы коллекции.

```python
for item in ["a", "b", "c"]:
    print(item)
```

### `range`

```python
for i in range(1, 6):
    print(i ** 2)
```

`range(start, stop)` идет до `stop`, не включая его.

### Вложенные циклы

```python
for row in range(3):
    for col in range(3):
        print(row, col)
```

Вложенные циклы полезны для таблиц, полей, матриц и координат.

### `while`

`while` работает, пока условие истинно.

```python
counter = 5

while counter > 0:
    print(counter)
    counter -= 1
```

### `break` и `continue`

```python
for x in range(10):
    if x == 5:
        break
    if x % 2 == 0:
        continue
    print(x)
```

- `break` полностью завершает цикл.
- `continue` пропускает текущий шаг и переходит к следующему.

## Функции

Функция помогает назвать повторяемое действие.

```python
def get_greeting(name):
    return f"Hello, {name}"

message = get_greeting("Bob")
```

Аргументы можно передавать позиционно или по имени.

```python
def describe_person(name, age):
    return f"{name}: {age}"

describe_person("Bob", 30)
describe_person(age=30, name="Bob")
```

`return` возвращает значение из функции. После `return` функция заканчивает выполнение.

## Области видимости

Переменные, созданные внутри функции, видны только внутри нее.

```python
def f():
    local_value = 10
    return local_value
```

Если функция использует внешнюю переменную, лучше передавать ее аргументом. Так код проще проверять и переиспользовать.

## Дата и время

```python
from datetime import datetime

start = datetime.now()
# какой-то код
end = datetime.now()

print(start)
print(end)
print(end - start)
```

## Частые ошибки

- Путать `=` и `==`: первое присваивает, второе сравнивает.
- Забывать, что индексы начинаются с `0`.
- Забывать, что правая граница среза не входит в результат.
- Менять список через одну переменную и удивляться, что изменения видны через другую.
- Писать `while` без изменения условия и получать бесконечный цикл.
- Забывать `return` в функции.
